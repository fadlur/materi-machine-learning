# ✅ Daily Checklist: Backend Developer → ML/AI Engineer Sprint

> **Cara pakai:** Copy template harian ke notes app favoritmu (Notion, Obsidian, Bear, atau bahkan sticky notes).  
> **Rule #1:** Checklist harian WAJIB diselesaikan sebelum tidur.  
> **Rule #2:** Kalau ketinggalan 1 item → jangan skip, tapi catat kenapa.

---

## 🌅 Morning Kickoff (08:00 - 08:30)

**Lakukan sebelum mulai coding. Ini set intention untuk hari ini.**

- [ ] **Wake up & light exercise** (10 min stretching/jalan) — otak butuh oksigen
- [ ] **Hydrate** — 1 gelas air, bukan kopi dulu
- [ ] **Review today's target** dari 90-day-action-plan.md — *"Hari ini saya harus selesai apa?"*
- [ ] **Set 3 Priorities** — tulis di sticky note / notes app:
  1. _________________________ (Must do — non-negotiable)
  2. _________________________ (Should do — important)
  3. _________________________ (Nice to do — kalau ada waktu)
- [ ] **Clear distractions** — phone silent, close unnecessary tabs, matikan notifikasi sosmed
- [ ] **Set timer** — Pomodoro 50/10 atau 90/15 (pilih yang cocok)

**Time budget: 30 menit max. Jangan over-plan.**

---

## 🧠 Deep Work Block 1: Teori & Learning (08:30 - 10:30)

**Fokus: Paham konsep. Jangan coding dulu.**

- [ ] **Baca / tonton materi** sesuai jadwal 90-day plan
- [ ] **Tulis catatan** — bukan copy-paste, tapi *rephrase dengan kata-kata sendiri*
- [ ] **Identifikasi 3 key takeaways** dari sesi belajar:
  1. _________________________
  2. _________________________
  3. _________________________
- [ ] **Tandai bagian yang confusing** — untuk ditanya nanti (forums, Discord, GPT)
- [ ] **Tidak cek sosmed / chat** selama blok ini

**Quality check:** Bisa jelaskan konsep tadi tanpa melihat notes? Kalau belum, ulang baca.

---

## ☕ Break (10:30 - 10:45)

- [ ] **Jauhkan dari layar** — lihat jauh, gerakkan tubuh
- [ ] **Snack / kopi** — jangan di meja komputer
- [ ] **No phone scrolling** — biar otak reset beneran

---

## 💻 Deep Work Block 2: Coding & Implementation (10:45 - 12:45)

**Fokus: Build. Jangan perfectionism, jalan dulu.**

- [ ] **Buka file yang kemarin ditinggal** — review 2 menit
- [ ] **Tulis 1-3 test cases / assertions** sebelum implementasi (kalau memungkinkan)
- [ ] **Implementasi** — follow exercise/challenge di modul
- [ ] **Run & debug** — kalau error, debug minimal 20 menit sebelum nyerah
- [ ] **Commit ke git** — pesan commit deskriptif: `"fase2: implement gradient descent with momentum"`
- [ ] **Catat blockers** — apa yang bikin stuck?

**Rule:** Kalau stuck >30 menit → catat, skip, lanjut ke item berikutnya. Jangan buntu di 1 tempat.

- [ ] **Makan tanpa layar** — biar otak digest informasi pagi
- [ ] **Light walk** — 10-15 menit
- [ ] **No coding thoughts** — biar subconscious processing bekerja

## 🏗️ Deep Work Block 3: Project Building (13:45 - 16:45)

**Output harian:** Harus ada "thing that works" — walau kecil. Bisa jadi 1 endpoint, 1 plot, 1 feature.

## 🚨 Emergency Protocols

### Kalau Hari Ini "Failed"

**Jangan panik. Semua orang punya bad day.**

1. **Tulis 1 hal yang berhasil** — walau kecil ("saya bangun pagi", "saya baca 1 halaman")
2. **Identifikasi 1 penyebab** — kenapa hari ini tidak produktif? (tidur buruk? distraksi? mood?)
3. **Jangan double-down besok** — jangan bikin target gila-gilaan untuk "compensate"
4. **Reset ke baseline** — besok ikutin checklist normal, jangan overwork
5. **Track pattern** — kalau 3 hari berturut-turut failed, ada yang perlu di-adjust

### Kalau Ketinggalan 3+ Hari

1. **Jangan skip ke modul sekarang** — lanjutkan dari yang ditinggal
2. **Potong scope** — fokus deliverable utama, skip challenge/optional
3. **Re-evaluasi timeline** — mungkin butuh 100-110 hari. Itu OK.
4. **Tanya diri sendiri** — apakah target terlalu ambitious? Adjust.

### Kalau Burnout Signs Muncul

**Tanda-tanda:**
- Tidak bisa fokus walau sudah minum kopi
- Semua kode terlihat sama, tidak ada progress
- Tidak excited lagi sama ML
- Tidur terganggu / mimpiin bug

**Solusi:**
1. **Istirahat 1 hari penuh** — no coding, no ML
2. **Reconnect dengan "why"** — kenapa mulai ini? Tonton video inspiratif
3. **Ganti environment** — coding dari cafe / coworking
4. **Social connection** — ngobrol dengan teman / komunitas
5. **Kurangi scope** — mungkin 90 hari terlalu intense, adjust ke 120

## 💡 Productivity Hacks

### The "2-Minute Rule"
Kalau ada task yang bisa selesai dalam 2 menit → **lakukan sekarang**. Jangan masukkan ke todo list.

### The "Eat the Frog"
Kerjakan task yang **paling tidak mau dikerjakan** di awal hari. Sisanya akan terasa mudah.

### The "Shutdown Ritual"
Tulis di notes: *"Tomorrow I will start with ______"* — ini offload anxiety dari otak ke paper.

### The "Git Commit Streak"
Target: **1 commit per hari** — walau cuma fix typo. Streak building = momentum building.

### The "Screenshot Habit"
Setiap ada progress yang visual → screenshot. Kumpulkan di folder "wins". Kalau down, lihat folder ini.
