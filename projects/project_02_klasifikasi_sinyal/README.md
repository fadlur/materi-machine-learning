# Project 2: Klasifikasi Sinyal / Sensor

## Tujuan
Klasifikasi status mesin/sistem berdasarkan sensor data.
Menggabungkan domain knowledge EE dengan ML.

## Deliverables
1. **Data Pipeline** — synthetic atau real sensor data
2. **Feature Engineering** — time + frequency domain features
3. **Model Comparison** — minimal 5 model berbeda
4. **Report** — analisis komprehensif

## Checklist
- [ ] Generate/acquire sensor data
- [ ] Time domain feature extraction
- [ ] Frequency domain feature extraction (FFT, spectral features)
- [ ] Domain-specific features (THD, power factor, crest factor, dll)
- [ ] Handle class imbalance
- [ ] Train & compare: LogReg, SVM, RF, GBM, KNN
- [ ] Hyperparameter tuning (GridSearch/RandomSearch)
- [ ] Feature importance analysis
- [ ] Confusion matrix + classification report
- [ ] Tuliskan INSIGHT domain knowledge!
